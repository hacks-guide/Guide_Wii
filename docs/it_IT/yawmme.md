# YAWM ModMii Edition

YAWM ModMii Edition è un'applicazione che consente di installare i file WAD e avviare applicazioni homebrew.
Questa guida mostrerà come installare i file WAD.

## Requisiti

- Una scheda SD o un'unità USB
- [YAWM ModMii Edition](https://oscwii.org/library/app/yawmme)

## Istruzioni

### Sezione I - Scaricamento

1. Scarica ed estrai il file `.zip` di YAWM ModMii Edition nella root della scheda SD o dell'unità USB.
2. Inserisci la scheda SD o l'unità USB nella console, poi avvia YAWM ModMii Edition dall'Homebrew Channel.

### Sezione II - Installazione dei file WAD

1. Seleziona il dispositivo che contiene i file WAD che desideri installare.

   ![](/images/homebrew/yawmME/source_device.png)

2. Naviga fino alla cartella contenente i file WAD.

   - Se hai una cartella chiamata `wad`, verrà aperta automaticamente.

   ![](/images/homebrew/yawmME/file_selection.png)

3. Naviga fino al file WAD che vuoi installare e premi `A`.

   - Se vuoi installare più file WAD contemporaneamente, premi `+` su ciascuno di essi, poi premi `A`.

   ![](/images/homebrew/yawmME/install_wad.png)

4. Premi ancora `A` per installare il o i file WAD.

   ![](/images/homebrew/yawmME/installing_wad.png)

   ![](/images/homebrew/yawmME/installing_wad_ok.png)
