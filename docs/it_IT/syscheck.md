# SysCheck

Un SysCheck elenca tutti gli IOS e i cIOS che sono nella console Wii, fornendo alcune informazioni riguardanti ciascuno di essi. Ti fornisce anche altre informazioni utili. Quando hai problemi relativi agli IOS, qualcuno potrebbe chiederti di eseguire un SysCheck.

## Requisiti

- Una scheda SD o un'unità USB
- [SysCheck ModMii Edition](https://oscwii.org/library/app/SysCheckME)

## Istruzioni

### Sezione I - Scaricamento/installazione

1. Estrai SysCheck ModMii Edition nella scheda SD o nell'unità USB.
2. Inserisci la scheda SD o l'unità USB nella console, poi avvia SysCheck ModMii Edition dall'Homebrew Channel.

### Sezione II - Esecuzione del SysCheck

1. Aspetta che il programma prenda le informazioni della console.

2. Quando la scritta `This IOS will be tested (Please select)` appare sullo schermo, premi `A`.

   ![](/images/homebrew/syscheck/syscheck_chooseios.png)

3. Aspetta che il programma prenda le informazioni degli IOS.

4. Quando ha finito, premi `A` se vuoi vedere cosa ha rilevato.

   ![](/images/homebrew/syscheck/syscheck_success.png)

5. Se vuoi condividere il SysCheck, puoi inviare il file `syscheck.csv` che si trova nella root della scheda SD o dell'unità USB. Puoi anche premere `A`, che invierà il resoconto al sito [syscheck.rc24.xyz](http://syscheck.rc24.xyz).
