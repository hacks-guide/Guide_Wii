    Trasferimento dati di salvataggio

    Per supporto in Inglese, chiedere aiuto sul server Nintendo Homebrew su Discord.

    Questa guida ti spiegherà come trasferire i dati di salvataggio dei giochi tra:

    • un Wii/Wii U (vWii),
    • Emulatore Dolphin,
    • memory card di GameCube
    • e memory card emulate (Nintendont).

    Di cosa hai bisogno
    • una scheda SD formattata come FAT32
    Istruzioni
    1. Inserisci la scheda SD nel tuo Wii/Wii U.
    2. Nel Menu Wii, seleziona Opzioni Wii dall’angolo in basso a sinistra.
    3. Selezione Gestione dati (se sei su Wii), Dati di salvataggio e Wii.
    4. Seleziona un salvataggio che vuoi trasferire, poi seleziona Copia.
      • Se il pulsante Copia non è selezionabile, dovrai installare Priiloader per disattivare la protezione della copia.
    5. Ripeti il passaggio precedente per ciascun salvataggio che vuoi trasferire.
    6. Una volta che hai finito, inserisci la scheda SD nell’altro Wii/Wii U.
    7. Nel Menu Wii, seleziona Opzioni Wii dall’angolo in basso a sinistra.
    8. Selezione Gestione dati (se sei su Wii), Dati di salvataggio e Wii.
    9. Seleziona la scheda Scheda SD dall’angolo in alto a destra.
    10. Selezione uno dei salvataggi che hai copiato alla tua scheda SD, poi seleziona Copia.
    11. Ripeti il passaggio precedente per ciascun salvataggio che hai copiato alla tua scheda SD.

    Assicurati di aver installato cIOS prima di seguire!

    Di cosa hai bisogno
    • L’Homebrew Channel su entrambi i sistemi
    • Una scheda SD o dispositivo USB
    • SaveGame Manager GX
    Istruzioni
    1. Estrai SaveGame Manager GX nel root della tua scheda SD o unità USB e inseriscila nel tuo Wii/Wii U.
    2. Avvia l’Homebrew Channel e avvia SaveGame Manager GX.
    3. Assicurati che la sorgente sia impostata su Wii.
    4. Trova e seleziona il tuo salvataggio, poi Extract e poi Default.
      • Ripeti questo passaggio per gli altri salvataggi che vuoi copiare.
    5. Premi il pulsante HOME ed esci da SaveGame Manager GX.
    6. Inserisci la scheda SD nel Wii/Wii U di destinazione.
    7. Crea un nuovo file di salvataggio per ogni gioco. Questo è necessario in modo che SaveGame Manager GX possa sovrascriverli.
    8. Avvia l’Homebrew Channel e avvia SaveGame Manager GX.
    9. Assicurati che la sorgente sia impostata su SD o USB.
    10. Seleziona ciascuno dei tuoi salvataggi e clicca Install.

    Importando il tuo backup della tua NAND su Dolphin, avrai effettivamente copiato tutto che c’era nel tuo Wii (al momento del backup). Questo ha anche il vantaggio di consentire l’accesso a Wiimmfi assieme ai canali patchati di RiiConnect24. Se l’hai già fatto, considera di utilizzare SaveGame Manager GX per risparmiare tempo.

    Di cosa hai bisogno
    • un backup della NAND
    • L’ultima versione beta dell’Emulatore Dolphin
    Istruzioni
    1. Dopo aver creato il tuo backup della NAND, inserisci la tua scheda SD al tuo PC e apri l’Emulatore Dolphin.
    2. Seleziona Strumenti > Gestisci NAND > Importa Backup NAND BootMii.
    3. Nella finestra di dialogo file che si apre, apri la tua scheda SD e seleziona nand.bin.

    Di cosa hai bisogno
    • L’Homebrew Channel sul tuo Wii/vWii
    • SaveGame Manager GX
    • Una scheda SD o dispositivo USB
    • L’ultima versione beta dell’Emulatore Dolphin
    Istruzioni
    1. Sul tuo Wii/Wii U, avvia l’Homebrew Channel e avvia SaveGame Manager GX.
    2. Trova e seleziona il salvataggio che vuoi trasferire, seleziona Extract e poi Default.
      • Ripeti questo passaggio per gli altri salvataggi che vuoi trasferire.
    3. Una volta estratti tutti i tuoi file di salvataggio, premi HOME ed esci da SaveGame Manager GX.
    4. Inserisci la tua scheda SD o unità USB nel tuo PC.
    5. Avvia l’Emulatore Dolphin.
    6. Seleziona Strumenti > Importa Salvataggio Wii....
    7. Nella finestra di dialogo file che si apre, vai nella cartella savegames della tua scheda SD o unità USB e seleziona ciascun file .bin al suo interno. Purtroppo, puoi selezionare solo un file alla volta, quindi dovrai ripetere l’ultimo passaggio per ciascun file .bin.

    Tutti i metodi elencati qui presuppongono che tu abbia l’ultima versione Beta o di sviluppo dell’Emulatore Dolphin.

    Di cosa hai bisogno
    • una scheda SD
    Istruzioni
    1. Avvia l’Emulatore Dolphin.
    2. Se stai esportando un salvataggio di un gioco, fai clic destro sul gioco nella finestra dell’Emulatore Dolphin e seleziona Esporta Salvataggio Wii.
      • Se stai esportando tutti i tuoi salvataggi Wii da Dolphin, seleziona Strumenti > Esporta tutti i Salvataggi Wii.
    3. Nella finestra di dialogo cartelle che si apre, seleziona la tua scheda SD. (Non entrare nelle altre cartelle!)
    4. Espelli la tua scheda SD in modo sicuro, quindi inseriscila nel tuo Wii/Wii U.
      • Se sei su Wii U, assicurati di avviare la modalità Wii.
    5. Seleziona il pulsante Opzioni Wii dall’angolo in basso a sinistra.
    6. Seleziona Gestione dati (se sei su Wii), Dati di salvataggio e poi Wii.
    7. Seleziona la scheda Scheda SD dall’angolo in alto a destra.
    8. Seleziona ciascun salvataggio che vorresti copiare al tuo Wii/Wii U e seleziona Copia.

    Di cosa hai bisogno
    • una Wii con le porte GameCube e l’Homebrew Channel installato
    • Una scheda SD o dispositivo USB
    • GCMM
    Istruzioni
    1. Avvia l’Emulatore Dolphin.

    2. Nel menu principale, seleziona File > Apri Cartella Utente.

    3. Nella cartella che si apre, vai su GC > [Regione salvataggi] > Card A.
    4. Copia i file .gci che vuoi trasferire sulla tua Memory Card.
    5. Crea una cartella di nome MCBACKUP nella tua scheda SD o unità USB e sposta i file .gci che hai appena copiato al suo interno.

    Se Dolphin è impostato per l’utilizzo di memory card .raw, vai su Strumenti > Memory Card Manager per esportare i tuoi salvataggi in .gci.

    1. Scarica ed estrai GCMM nel root della tua scheda SD o unità USB.
    2. Espelli la tua SD/USB dal tuo PC in modo sicuro, quindi inseriscila nel tuo Wii.
    3. Avvia l’Homebrew Channel e avvia GCMM.

    4. Seleziona il dispositivo dove sono stati copiati i file .gci.

    5. Premi +/X per procedere con la modalità di ripristino.

    6. Seleziona lo slot che contiene la tua Memory Card.

    7. Seleziona il salvataggio che vuoi ripristinare.
      • Se vuoi ripristinare tutti i salvataggi all’interno di MCBACKUP, premi 1/R.
    8. Appena completato il ripristino, premi un pulsante qualsiasi per continuare.
    Istruzioni
    1. Apri l’Emulatore Dolphin e seleziona Strumenti > Memory Card Manager dal menu principale.
    2. Clicca il pulsante Apri che è accanto allo Slot B.
    3. Vai nella cartella saves nella tua scheda SD o unità USB e seleziona il file .raw con il nome dell’ID del gioco.
      • Se non hai già creato un salvataggio con Nintendont, seleziona Crea e imposta la dimensione della Memory Card a 251 blocchi.
      • Se utilizzi una Memory Card per tutti i giochi, apri il file ninmem.raw/ninmemj.raw dal root della tua SD/USB.
    4. Se Dolphin archivia i tuoi salvataggi in una cartella GCI (per predefinito):
      • Clicca Scambia a B.
      • Clicca Importa e vai su <REGION>/Card A/.
      • Seleziona il (oppure i) file .gci che vuoi importare. Altrimenti, se Dolphin archivia i tuoi salvataggi in file .raw, semplicemente seleziona i tuoi salvataggi dal lato sinistro e seleziona Copia su B.

    Tutti i metodi elencati qui richiedono un Wii con porte GameCube.

    Di cosa hai bisogno
    • GCMM
    Istruzioni
    1. Scarica ed estrai GCMM nel root della tua scheda SD o unità USB.
    2. Espelli la tua SD/USB dal tuo PC in modo sicuro, quindi inseriscila nel tuo Wii.
    3. Avvia l’Homebrew Channel e avvia GCMM.

    4. Seleziona il dispositivo su cui copiare il file di salvataggio.

    5. Premi B e -/L eY per entrare in modalità backup Raw.

    6. Seleziona lo slot che contiene la tua Memory Card.

    7. Appena completato il dump, premi un pulsante qualsiasi per continuare.
    8. Premi HOME/START per uscire da GCMM.
    9. Inserisci la tua scheda SD o unità USB nel tuo PC.
    10. Copia il file .raw all’interno di MCBACKUP nel root del dispositivo utilizzato per Nintendont e rinominalo in ninmem.raw.
      • Se i tuoi giochi sono giapponesi, Nintendont userà ninmemj.raw perché i giochi giapponesi non funzioneranno tanto bene quando sono presenti salvataggi di altre regioni.
    11. Avvia Nintendont e imposta l’opzione Memcard emulation a Multi [ ? ]
